# Projeto Cordel Moderno

Este projeto é uma página da web criada para apresentar o poema "Cordel Moderno" de Milton Duarte, com um estilo que remete à literatura de cordel. A página utiliza um efeito de paralaxe para tornar a leitura mais dinâmica e visualmente agradável.

## O Poema

O poema "Cordel Moderno", de autoria de Milton Duarte, reflete sobre como a tecnologia e a internet transformaram a comunicação e a linguagem no dia a dia.

*   **Autor:** Milton Duarte
*   **Fonte Original:** [Recanto das Letras](https://www.recantodasletras.com.br/poesias/3186743)

## Tecnologias

*   HTML5
*   CSS3
